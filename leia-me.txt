Apresentação
Foi disponibilizado o arquivo INF1301-Trab3-FVLCVN.zip. Nele há arquivos do modulo lista, peça, tabuleiro, jogo e main para o jogo de Xadrez.

Instalação
Separe uma pasta vazia para instalar o jogo e extraia os arquivos do .zip para ela.
Nesse diretório haverá todos os arquivos necessário para rodar o programa.
Caso queira criar uma peça nova para o xadrez deve-se digitar no PecasNovas.txt no seguinte padrão:
	(Identificação 1)		 (Cor)
	(MovimentoX0)		(Movimento Y0)
	(MovimentoX1)		(Movimento Y1)
	.
	.
	.
	(MovimentoXF)		(Movimento YF)
	(Identificação 2)		  (Cor)
	.
	.
	.
	(Identificação 3)		 (Cor)


Estrutura de diretórios

Executáveis: TesteLista.exe
Batches: Final.bat

-Peça
Código Fonte: peca.c, peca.h e testePeca.c
Script De Teste: testaPeca.script
Executaveis: TestePeca.exe
Batches: Final.bat

-Tabuleiro
Código Fonte: tabuleiro.c, tabuleiro.h e testeTabuleiro.c
Script De Teste: testeTabuleiro.script
Executáveis: TesteTabuleiro.exe
Batches: Final.bat

-Jogo
Código Fonte: Jogo.c, Jogo.h e testeJogo.c
Script De Teste: testeJogo.script
Executáveis: TesteJogo.exe
Batches: Final.bat

-Main
Código Fonte: main.c

Documentação
ModeloFisico.pdf: É um arquivo que ilustra o modelo físico do jogo
Assertivas Estruturais: É um arquivo que define algumas assertivas da arquitetura do programa.
Leia-Me.txt: É um arquivo que possui a explicação de como utilizar o programa  em questão
ManualDoJogador.txt:É um arquivo que possui o tutorial de como jogar o xadrex
Arquitetura da Aplição: É um arquivo descrevendo a arquitetura utilizada para confeccionar o trabalho
Exemplo de modelo: Ilustra os modelos ultilizados no trabalho
-Execução dos Testes
Os testes dos arquivos desse programa são executados no CMD do Windows.
Testando os arquivos do código fonte:
 	Abrir o CMD
	Digite cd C:/.../nomeDaPastaEmQuestão
	Digite Final.bat
	OBS: Cada diretório testará apenas os códigos fonte dentro dela 

-Executar o Xadrex
Clique no main.exe o jogo abrirá
